*CLI tools and TUI apps that I use regularly.*

Terminal apps make using a computer more efficient:
- **Speed** — no heavy GUI framework
- **Flexibility** — chaining commands like searching, filtering, and counting lines is trivial on the command line
- **Customization** — keybindings, layouts, looks
- **Scripting** — anything you can do on the terminal, you can automate

## The Collection

[abook](https://abook.sourceforge.io/) text-based addressbook

[calcurse](https://www.calcurse.org/) calendar and scheduling application

[glow](https://github.com/charmbracelet/glow) render markdown on the CLI

[gurk](https://github.com/boxdot/gurk-rs) Signal Messenger client for terminal  

[htop](https://htop.dev/) terminal process viewer

[hugo](https://gohugo.io/) website building framework

[iamb](https://github.com/ulyssa/iamb) terminal client for Matrix with vim-bindings

[marks](https://github.com/kullbachxyz/marks) minimal ncurses-style bookmark manager 

[newsboat](https://github.com/newsboat/newsboat) RSS feed reader for the terminal 

[pulsemixer](https://github.com/GeorgeFilipkin/pulsemixer) CLI audio mixer

[rmpc](https://github.com/mierak/rmpc) terminal-based MPD client

[neovim](https://neovim.io/) hyperextensible Vim-based text editor

[yazi](https://github.com/sxyazi/yazi) terminal file manager
